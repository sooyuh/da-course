🤖 AI时代Python数据分析训练项目

适配Cloudflare Pages · 纯静态无后端 · 含完整可运行示例

本方案包含10个数据分析训练项目，涵盖购物车分析、聚类、RFM、关联规则、时间序列等核心技能。所有项目均为纯静态实现，可零成本部署到Cloudflare Pages，学生可直接在浏览器中练习，也可参考Python代码在本地运行。

📚 十大训练项目清单

每个项目均提供：数据集字段说明、分析任务、涉及pandas技术点。

项目 1

🧹 销售数据清洗与缺失值处理

数据字段：订单ID、日期、金额、客户ID、地区
任务：识别缺失值并合理填充、删除重复订单、日期标准化、输出清洗摘要。
技术点：read_csv()、dropna()、fillna()、drop_duplicates()、to_datetime()

项目 2

🛒 购物车放弃率分析（核心）

数据字段：用户ID、加入时间、是否完成支付、商品总额、商品数量
任务：计算放弃率；按商品总额分箱分析放弃率变化；对比放弃/支付用户的商品数量差异。
技术点：groupby、cut()分箱、分组聚合

项目 3

👑 用户复购RFM分析

数据字段：订单ID、用户ID、订单日期、金额
任务：计算R(最近消费)/F(频次)/M(金额)，分档并标记高价值用户，输出TOP10。
技术点：groupby+agg、merge、分档函数

项目 4

🔗 商品关联规则挖掘（购物篮分析）

数据字段：订单ID、商品名
任务：构建订单×商品矩阵，计算支持度/置信度/提升度，发现强关联规则。
技术点：pivot_table、crosstab、Apriori算法逻辑

项目 5

🤝 用户消费行为聚类（K-Means）

数据字段：用户ID、月均消费金额、月均购买次数、折扣利用率、最近购买天数
任务：数据标准化、手肘法确定K值、K-Means聚类、分析各簇特征（高消费型/沉睡型等）。
技术点：KMeans、StandardScaler、聚类可视化

项目 6

📈 时间序列分析——日销售额趋势与异常检测

数据字段：日期、销售额、订单量
任务：绘制趋势图、移动平均±2倍标准差检测异常、分析“周几效应”、预测下一天销售额。
技术点：resample、rolling、shift

项目 7

🧩 多表合并与用户生命周期价值(LTV)计算

数据文件：users.csv, payments.csv, clicks.csv
任务：合并三表，求首单时间、总付费、7日/30日LTV，按渠道对比LTV差异。
技术点：merge、日期差计算、cumsum

项目 8

😊 文本情感分析 + 好评差评分组统计

数据字段：评论ID、评分(1-5)、评论文本
任务：基于关键词规则构建情感极性，计算情感分与评分一致率，分组统计情感词频。
技术点：apply、str.contains、value_counts

项目 9

🧪 A/B测试结果分析

数据字段：用户ID、组别A/B、是否转化、浏览时长
任务：计算转化率及置信区间、t检验显著性、按时长分层比较。
技术点：groupby统计、假设检验逻辑、cut分层

项目 10

🚀 全流程电商购物车预测（综合）

数据字段：用户属性、浏览商品数、加购时间、促销敏感度、是否购买
任务：完整清洗、特征工程（是否周末加购、加购到离开时长）、逻辑回归预测、输出特征重要性及混淆矩阵。
技术点：get_dummies、train_test_split、LogisticRegression、confusion_matrix

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
# from sklearn.cluster import KMeans  等机器学习库

# ==========================================
# 示例：项目2购物车分析完整代码
# ==========================================

# 1. 读取数据
df = pd.read_csv('cart_data.csv')
print("数据预览:")
print(df.head())
print("\n数据形状:", df.shape)

# 2. 计算整体放弃率
abandon_rate = (df['is_paid'] == 0).mean() * 100

# 3. 按商品总额分箱
df['amount_group'] = pd.cut(
    df['total_amount'],
    bins=[0, 100, 300, 10000],
    labels=['低额', '中额', '高额']
)

# 4. 分组统计
group_stats = df.groupby('amount_group')['is_paid'].mean()
group_abandon = (1 - group_stats) * 100

# 5. 输出结果
print(f"\n整体放弃率: {abandon_rate:.2f}%")
print("\n各分组放弃率:")
print(group_abandon)

# 6. 可视化（可选）
plt.figure(figsize=(10, 6))
group_abandon.plot(kind='bar', color='#4f46e5')
plt.title('购物车放弃率 - 按金额分组')
plt.ylabel('放弃率 (%)')
plt.xticks(rotation=0)
plt.tight_layout()
plt.show()
